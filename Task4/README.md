# Task4

This project contains FastAPI code for Task 4.

## How to Run

1. Install dependencies:
   ```bash
   pip install -r ../requirements.txt
   ```
2. Start the server:
   ```bash
   uvicorn main:app --reload
   ```

## Files

- `main.py`: Main FastAPI application.

## Notes

- Make sure Python 3.11+ is installed.
